# FolhaDePagamento

----------------------- MENU -----------------------
1 - Adicionar empregado;
Caso seja selecionada a opção “1”, o programa chamará o método “adicionarEmpregado”, passando o array de Empregados, a quantidade de empregados existentes.
Dentro do método será lido cada atributo da classe Empregado.

2 - Remover empregado;
Caso seja selecionada a opção “2”, o programa chamará o método “removerEmpregado”, passando o array de Empregados, o nome do empregado que será removido e a quantidade de empregados existentes.
É passado um “for” para achar o empregado que deseja ser removido, caso encontrado, ele é removido, senão, o programa emitirá uma mensagem “Empregado não encontrado”.

3 - Lançar um cartão de ponto (ainda não finalizada);

4 - Vendas;
No caso, essa função seria a "Lançar um resultado venda", contanto, ele poderá escolher entre registrar a venda do empregado e ver os produtos vendidos pelo empregado(que
no caso seria o resultado venda).

5 - Lançar uma taxa de serviço (ainda não finalizada);

6 - Alterar detalhes de um empregado;
Será chamado o método "alterarEmpregado" e dentro do método, será informado as opções para ser escolhido o que o usuario editará do empregado(nome, endereço etc).

7 - Rodar a folha de pagamento para hoje (ainda não finalizada);

8 - Undo/Redo(Parcialmente finalizada);
A opção de Undo funcionando paga algumas funcionalidades do menu.

9 - Agenda de Pagamento (Ainda não finalizada);

10 - Criação de Novas Agendas de Pagamento (Ainda não finalizada);

Obs : Projeto feito obrigatoriamente sem alguns conceitos de Orientação à objetos.
